актуальная дата 20.05.2024
рабочая ОС - Windows 8, 10

Часть 1. Назначение приложения.
Данное приложение предназначено для редактирования в байтовом режиме различных файлов компьютера.

Часть 2. Начало работы в приложении
Для начала работы необходимо открыть файл соответсвующей кнопкой в интерфейсе. После этого откроется первая страница-таблица содержащая значения байт в 16-ной системе счисления. Крайний левый столбец - начало сдвига строки в файле, верхняя строка - сдвиг байта в строке. Сложив сдвиг строки в файле и сдвиг байта в строке получается сдвиг байта в файле.

Часть 3. Механизм выделения областей в таблице.
Приложение позволяет выделять байтовые отрезки в таблице. Для этого предусмотрен механизм нажатия правой кнопки мыши по ячейке байта в таблице, который работает по клику ПК мыши по строке с числовым значением (не **). Механизм имеет три состояния:
0 - в самом начале работы, когда не было кликов
1 - кликнули по ячейке-началу области
2 - кликнули по ячейке-концу области
3 - сбросили выделение

Часть 4. Блок фокус-выделение.
Блок содержит данные о текущем размере таблицы в ячейках. Размеры таблицы можно регулировать нажатием специальной кнопки в интерфейсе. Нужно помнить, что одновременно на странице-таблице может быть не более 1 МБ данных.
Поле положение курсора указывает на сдвиг байта, на который установлен фокус в таблице.
Начало выделения указывает на стартовый сдиг выделенной области в таблице, Длина выделения указывает на длину выделенной области.

Часть 5. блок файловый менеджер.
Позволяет открывать новые файлы, сохранять копии текущего под новыми именами.

Часть 6. Блок Поиск.
Приложение позволяет выполнять поиск в байтах файла по десятичному беззнаковому значению или по двоичной маске (строгие значения байтов - 0 и 1, нестрогие - *, например 0011**00). Можно искать блоки данных от 1 до 8 байт.

Часть 7. Блок редактирование.
Приложение позволяет выполнять операции удаления и вырезки со смещением или обнулением области, вставки значений из буффера или из строки с значениями байт в шестнадцатиричном виде через ; с замещением или сдвигом, можно вставлять нули.
Для выполнения операции необходимо выделить определённую область (в операциях где важна старотовая позиция важно наличие в выделенной области этой старотовой позиции)

Часть 8. Блок знчений по блокам

В нём приведены целые значения с/без знака, значения с плавающей точкой одинарной и двойной точности для блоков 1, 2, 4 и 8 байт.
